# TypeScript CLI Tool Boilerplate

## Commands

Run development mode

This is the same as parallel execute of `npm run build:dev` and `npm run start:dev`

```bash
npm run dev
```

Production build

```bash
npm run build
```

Debug build with watch

```bash
npm run build:dev
```

Run production build

```bash
npm start
```

Run debug build with watch

```bash
npm run start:dev
```

Run test

```bash
npm run test
```

Run type check

```bash
npm run tc
```
