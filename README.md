# TypeScript CLI Tool Boilerplate

## Commands

Run development mode

This is the same as parallel execute of `npm run build:dev` and `npm run start:dev`

```bash
npm run dev
```

Production build

```bash
npm run build
```

Run production build

```bash
npm start
```

Run test

```bash
npm run test
```

Run test once

```bash
npm run test:once
```

Run type check

```bash
npm run tc
```
